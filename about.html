About the Project

This website is a simple educational resource about Deadlock, a team-based game by Valve. It is made for students and new players who want a clear starting point.

Learning Goal

The goal is to explain basic game concepts without long guides. Each page focuses on a different learning activity: reading, watching, comparing resources, or answering a quiz.

How to Use It

Start with the lessons page, watch the short training media, review the resource table, and finish with the quiz form.
